# 🏭 Calculateur de plan d'usine Satisfactory

Ce projet est un calculateur visuel pour planifier automatiquement des usines optimisées dans le jeu **Satisfactory**.  
Vous entrez les ressources brutes disponibles et la ressource finale souhaitée : le script vous calcule instantanément le plan d'usine optimal, affiche toutes les machines nécessaires et génère le schéma de production !

## ✨ Fonctionnalités

- Choix de la ressource à produire (ingots, structures, etc)
- Saisie des stocks de ressources brutes (fer, cuivre, charbon, pétrole…)
- Génération automatique du plan complet avec machines, inputs, outputs, rendements, surplus
- Visualisation schématique du flux de production (arbre de production interactif)
- Alertes en cas de manque ou de surplus de ressources

## 🚀 Utilisation

1. **Téléchargez le repo** (bouton vert “Code” > Download ZIP ou via `git clone`)
2. Placez tous les fichiers dans un même dossier (notamment `Optimizer Satisfactory.html` et le dossier `/images`)
3. **Ouvrez le fichier `Optimizer Satisfactory.html` dans votre navigateur** (double-clic ou clic droit “Ouvrir avec”)
4. C’est prêt ! Pas besoin de serveur, tout fonctionne en local.

## 💡 Conseils

- Compatible PC/Mac (navigateur moderne recommandé)
- Pour une utilisation mobile, hébergez le projet sur [GitHub Pages](https://pages.github.com/) ou un service web statique
